    ❓ Comment lire les scores (détails complets)

    Signe = direction : vert = LONG (hausse), rouge = SHORT (baisse).

    Grandeur = force de conviction (et non un % de mouvement attendu). Probabilité ≈ 50% + |score|/15, plafonnée à 100%.

    Repères :

    • |score| ~0 (<0.05, ⚪) → ~50%, pile ou face, non-actionnable
    • |score| ≈ 1,5 → ~60% de conviction
    • |score| ≈ 3,75 → ~75% de conviction
    • |score| ≥ 7,5 → 100%, conviction maximale

    Synthèse des décisions (en haut du bulletin) = vue d'oiseau : direction + note (le score signé, ex. SHORT -3.82 ; plus |note| est élevé, plus la conviction est forte) ou 🚫. Le détail chiffré complet est dans la Matrice plus bas.

    [pond: …] = score en version pondérée (news × matérialité × fiabilité), affiché seulement s'il diffère du primaire. Sur les cellules 📰 (news>50%), le pondéré (tempéré, plus fiable) passe en tête et le brut est entre parenthèses : LONG +3.77 (brut +5.69) 📰.

    Symboles : 🚫 données insuffisantes · ⚑ gate · 📰 news>50% du quant · ⚪ quasi coin-flip non-actionnable · ⚠ divergence primaire/pondéré. La légende complète du bulletin ne liste que les symboles réellement présents.
